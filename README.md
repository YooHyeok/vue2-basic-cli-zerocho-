# 게임별 프로젝트 관리

## 프로젝트 초기화
```bash
npm init
```

```bash
package name : 
```
ex) 숫자 야구 게임 : `number-baseball` 입력

이후 옵션은 모두 Enter로 Skip.
프로젝트 초기화 작업이 종료되면 package.json 파일이 생성된다.

## vue 설치
```bash
npm install vue
```
## webpack 설치
```bash
npm install webpack webpack-cli -D
```
